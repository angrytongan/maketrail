# Physical build plans (not yet reviewed)

Source: https://www.lunchridemtb.com/plans

These are downloadable PDF blueprints for real, buildable wooden trail features — different from the parametric/physics sources in [[jumps]], [[berms]], [[rollers]] (which cover *geometry rules*), these cover *literal construction* (bill of materials, joinery, angles). Worth reviewing once we get to modeling obstacle geometry in 3D, since real hand-built ramps are a reasonable reference for what a "kicker" shape actually looks like in practice. None of these have been read yet — download + OCR later, same as [[sources-to-review]].
